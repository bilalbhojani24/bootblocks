What is BootBlock?

Bootblock is a collection of different layout of each sections which are used in websites. Bootblock boost designing and developing part of your website. The layout which is provided are 100% mobile-responsive layouts.

CDN to use bootblock code

How to use

Step 1 : Choose the structure/layout of your choice.

Step 2 : Below the layout, there are two buttons preview and copy to clipboard. Use preview button to view the code and copy to clipboard to copy the code.

Step 3 : Semple, paste the code to your file and get output.
